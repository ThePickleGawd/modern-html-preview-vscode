# Modern HTML Preview

Open an `.html` or `.htm` file and press `cmd+shift+v` on macOS or `ctrl+shift+v` elsewhere.

Better than traditional HTML preview because it serves the file from an HTTP origin instead of injecting HTML into a webview. That means inline scripts, ES modules, relative `fetch()`, CSS, images, fonts, anchors, forms, unsaved edits, and asset reloads behave much more like a real browser.
